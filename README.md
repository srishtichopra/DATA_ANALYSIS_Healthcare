# MediTrack-Appointment-Analysis

## Overview
Analysis of 110,000+ medical appointment records to understand
why patients miss their scheduled appointments.

## Dataset
Kaggle — Medical Appointment No Shows (Brazil, 2016)
110,527 records 

## What's Covered
- Data Cleaning (negative age, duplicate columns, wait days)
- Exploratory Data Analysis (EDA)
- Age group vs disease prevalence
- Impact of SMS reminders on attendance
- Gender, Scholarship, Hypertension, Diabetes analysis

## Key Findings
- Female patients have taken more appointments then male patients
- Patients without SMS reminders show up more (84%) than those who received one (72%)
- Hypertension patients are more likely to show up (85%)
- Longer wait times = higher no-show rate
- Younger patients (20-30) have the highest no-show rate

## Tech Stack
Python | Pandas | Matplotlib | Seaborn
